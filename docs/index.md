# Wine Quality ML Project

Проект по предсказанию качества вина с использованием ML.

## Возможности

- ✅ DVC Pipeline для воспроизводимости
- ✅ Pydantic для валидации конфигураций
- ✅ ClearML для трекинга экспериментов
- ✅ W&B для визуализации
- ✅ Автоматизация обучения

## Архитектура
````
data/ → prepare → train → evaluate → model
````
